# webapp

> vue with cordova

## Build Setup

```bash
# 安装依赖
npm install

# 在开发环境下热更新调试
npm run dev

# 在浏览器端运行
## 引入了cordova.js并监听deviceready事件
npm run broswer

# 在android设备上运行
npm run android

# 构建到cordova目录下
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
